# otp-enter-into-an-blank-box
this is a simple c program in which i try to describe the OTP management system of the blank box .

THE Features of the the code:-
1.here first entry the OTP.
2.The code first generate a verification box and the enter the OTP 
inside the box.
3.The verification box recognise that OTP,for the authentication.
4.when the OTP is verified then it will give the results as yes or no.
5.from this code website can easily knowing that the OTP is authorised or not.
